# voclab
Tool to look up und learn words in other languages

## APK herunterladen / Download APK

Die neueste APK-Datei kann direkt von GitHub Actions heruntergeladen werden:

1. Gehe zu **[Actions](../../actions/workflows/build-apk.yml)** → wähle den neuesten erfolgreichen Build-Lauf aus.
2. Scrolle ganz nach unten zum Abschnitt **Artifacts**.
3. Lade **voclab-debug** herunter und entpacke die ZIP-Datei.
4. Installiere die `app-debug.apk` auf deinem Android-Gerät (USB-Debugging oder Dateiübertragung).

> **Hinweis:** Stelle sicher, dass auf deinem Gerät unter *Einstellungen → Sicherheit* die Option **"Unbekannte Quellen"** (bzw. *"Apps aus unbekannten Quellen installieren"*) aktiviert ist.

---

_To download the APK, navigate to the [Actions tab](../../actions/workflows/build-apk.yml), open the latest successful run, and download the **voclab-debug** artifact._
